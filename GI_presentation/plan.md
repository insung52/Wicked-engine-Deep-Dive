수업 자료 만들기

vizmotive 엔진에 적용한 3개의 GI 를 설명하는것이 목표

설명의 수준은 코드 수준까지는 필요없고, 작동 흐름 위주로 (ddgi 예시 : probe 들에서 매 프레임 ray 발사 -> 이후 무한 반사가 어떻게 작동되는지까지)

vizmotive 엔진의 실제 작동 화면을 스크린샷으로 사용하여 발표 자료로 사용

총 3개의 발표자료

1. DDGI

- 첫 자료이니 GI 에 대한 개념 설명도 초반에 필요
- DDGI 의 작동 원리 설명

2. VoxelGI

- VoxelGI 의 작동 원리 설명

3. SurfelGI

- SurfelGI 의 작동 원리 설명
